# 本地服务器使用说明

## 快速启动

### 方法1：双击启动（推荐）

**Windows用户：**
- 双击 `start-server.bat` 文件
- 服务器会自动启动，并在浏览器中打开页面

**PowerShell用户：**
- 右键点击 `start-server.ps1`，选择"使用PowerShell运行"

**Python用户：**
- 双击 `start-server.py` 文件（如果已配置Python关联）
- 或在命令行中运行：`python start-server.py`

### 方法2：命令行启动

#### 使用Python（推荐，最简单）

```bash
# 在项目目录下运行
python -m http.server 8000
```

然后在浏览器中打开：`http://localhost:8000/A1.html`

#### 使用Node.js

```bash
# 安装http-server（只需安装一次）
npm install -g http-server

# 启动服务器
http-server -p 8000 -o /A1.html
```

## 服务器地址

启动后，在浏览器中访问：
- **主页面**: http://localhost:8000/A1.html
- **根目录**: http://localhost:8000/

## 停止服务器

在运行服务器的命令行窗口中，按 `Ctrl + C` 停止服务器。

## 常见问题

### 1. 端口被占用

如果8000端口被占用，可以修改端口号：

**Python:**
```bash
python -m http.server 8080
```

**Node.js:**
```bash
http-server -p 8080
```

然后访问：`http://localhost:8080/A1.html`

### 2. Python未安装

下载安装Python：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### 3. Node.js未安装

下载安装Node.js：https://nodejs.org/
- 选择LTS版本（长期支持版）

### 4. 防火墙提示

如果Windows防火墙提示，选择"允许访问"即可。

## 为什么使用本地服务器？

1. **解决CORS问题**：避免浏览器的跨域限制
2. **更好的路径处理**：正确处理相对路径和图片加载
3. **调试方便**：可以使用浏览器开发者工具完整调试
4. **模拟真实环境**：更接近实际部署环境

## 技术说明

- **Python服务器**：使用Python内置的`http.server`模块，无需额外安装
- **Node.js服务器**：使用`http-server`包，功能更丰富
- **端口**：默认使用8000端口，可在脚本中修改

## 推荐配置

- **开发测试**：使用Python服务器（简单快速）
- **生产环境**：使用专业的Web服务器（如Nginx、Apache等）
