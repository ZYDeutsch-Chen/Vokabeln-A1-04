# A1 德语名词卡片学习系统

这是一个用于学习德语 A1 级别名词的交互式词汇卡片系统。

## 功能特点

- 📚 13 个主题分类，共 382 个 A1 级别名词
- 🎴 交互式卡片翻转学习
- 📊 学习进度跟踪
- 🎯 随机练习模式
- 📱 响应式设计，支持移动设备
